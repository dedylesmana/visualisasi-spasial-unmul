# Interactive Spatial Visualization of Universitas Mulawarman Campus

An interactive geospatial and mapping project for campus facilities at Universitas Mulawarman, developed using Python, Folium, and GeoPandas. This project serves as a real-world data science portfolio demonstration.

## 🌐 Live Interactive Demo
👉 **[Click Here to View the Interactive Map](https://dedylesmana.github.io/visualisasi-spasial-unmul/peta_interaktif_kampus_unmul.html)**

## 🚀 Features
- **Interactive Markers:** Detailed information on campus buildings and facilities upon clicking.
- **Geospatial Plotting:** Accurate coordinate boundary handling using GeoPandas.
- **Clean Interface:** User-friendly web map deployment via Folium.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Folium, GeoPandas, Pandas
